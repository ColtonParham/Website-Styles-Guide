Text Styles

Here is a list of the kinds of styles that were used to create this website. Primary font used is Josefin Sans

Font sizes used: include:

  • 21PX
  • 16PX
  • 32PX
  • 50PX

H1

This is the primary largest text style, used for header to head tag, and is the largest size.

Header

This tag will allow us to insert text, or an image at the highest possible point on the webpage.

H2

This is similar to the H1 tag, it does the same thing as an H1 tag, but is by default a smaller size.

P

This is the paragraph tag, it's used for larger pieces of information. We use it quite a bit here. Primarily used for each entry inside the elements.

Div

This is a Divider, it divides different sections of the page to whatever we deem is needed. Typically to indicate seperated groups for different information.

Body

The body is where everything is written, or for lack of better terms, displaced.
